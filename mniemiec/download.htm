Life RLE-to-text file converter

RLE2TXT is a utility that converts .RLE-format Life patterns used in these archives into human-readable text files.

Download the program in either of the following forms. It may be freely distributed. This can be compilable under MinGW under Windows, but should easily work with little or no change on most other operating systems:

  • A zip file containing a Windows-32 console executable, support files, and Gnu C++ source (9K).
  • A zip file containing Gnu C++ source and support files, but without any executable (4K).
