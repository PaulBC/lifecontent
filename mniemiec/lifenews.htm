Breaking news

The latter half of 2014 has seen a major explosion in the object synthesis area, so there has not been time to update these pages to keep up with them. This is a list of some of these discoveries, which will be added here as time permits:

  • Many oscillators and syntheses have improved syntheses.
  • All still-lifes up to 18 bits are proven to have syntheses.
  • All flotillae of gliders with arbitrary spacings are proven to have syntheses, which makes all the syntheses on these pages involving multiple gliders arriving from the same direction robust. (Syntheses involving other kinds of spaceships flying in parallel already explicitly show syntheses of those spaceships.)
  • Syntheses of many exotic spaceships, such as Crab, Dart, Weekender, and X66.
  • A new fundamental puffer, the Pufferfish, that can be synthesized, and can also be turned into a pure spaceship by adding two escort HWSS.
  • New Geminoid spaceships

Features

These are some of the main features of these pages:

  • Complete counts of Life objects up to 21 bits, with some up to 25 bits.
  • Complete lists of most of the above objects.
  • Syntheses for most objects up to 21 bits (15 for still-lifes, 16 for pseudo-still-lifes, 18 for period 2 oscillators, 17 for period 2 pseudo-oscillators), and some up to 25 bits, plus selected larger objects.
  • Above lists are arranged by object type, size in bits, cost in gliders, and frequency of occurrence, and also by name for common objects.
  • Similar object and synthesis data for several cellular automata rules other than Life.
  • All objects in the above lists can be directly searched by pattern image, name, and/or criteria such as population, period, velocity, cost in gliders, and many others.
  • All synthesis files for any given stamp image can be downloaded in a single zip file, for convenience.

Deficiencies

The following are known deficiencies and limitations with these pages:

  • While most object lists have syntheses for most objects on those lists, there are a small number of objects for which syntheses have been attempted, but no suitable synthesis methods have yet been found. These await better technology, inspiration, or blind luck.
  • Syntheses are only partially complete for 15-bit still-lifes and 18-bit period 2 oscillators.
  • Object list and syntheses are only partially complete for 21-bit period 3 oscillators.
  • Many lists of objects from 22-25 bits have been counted, but specific object lists have not yet been assembled, nor syntheses attempted for most of the objects, other than a superficial examination to determine which objects would be easy to synthesize using standard methods, and which ones would require special treatment, or not have syntheses at all.
  • Overweight spaceship flotillae have been counted, but not explicitly listed nor synthesized. Syntheses have been developed for most of them from one side or other, and most of these can be combined in pairs, like building blocks, to create the full overweight spaceship flotillae, but this has not been done yet.
  • Large lists (typically those involving between 1000 and 10000 objects) have large stamps of 1000 objects per page, but one cannot click on individual objects to see them in detail. These include still-lifes (of 16, 17 bits) and 18 bits), pseudo-still-lifes (of 17 and 18 bits), period 2 oscillators (of 19, 20 and 21 bits), period 2 pseudo-oscillators (of 18, 19, 20 and 21 bits).
  • Some lists consist entirely (or almost entirely) of a large number of objects for which syntheses are unknown, and are extremely unlikely to ever be found. Such lists sometimes omit clickable object files for objects without syntheses, especially if these objects occur in only that one list and nowhere else. These include several lists for the B2/S2 rule (still-lifes, period 2 oscillators and billiard tables). This does not apply to most explicit list of objects without syntheses as these contain objects that already exist on other lists.)

Bugs

The following are known bugs on these pages. These are specifically things that are proper HTML, and rendered correctly in most browsers, especially Chrome, Firefox and Safari, but which displayed incorrectly in Internet Explorer versions at least up to 10:

  • Sometimes, in tables with objects of different widths, Internet Explorer displays incorrect widths for some objects. The objects are in the correct relative positions, but the shapes of their boxes may be distorted.
  • Internet Explorer sometimes displays boxes for some exotic but valid Unicode characters. This affects some mathematical symbols on the Surreal Numbers and Games page, and its related calculator page.

Recent changes to these pages

This is a list of major changes in these pages. Minor fixes, such as fixed typos, re-worded descriptions, or slightly-improved syntheses (usually as a result of improved bookkeeping) are not listed:

2015-02-19 Major site overhaul, including the following, among others:

  • Total pages: The total number of distinct web pages has almost doubled from 65 to 128.
  • Total stamps: The total number of pattern stamp collections has almost tripled from 259 to 736.
  • Total files: The total number of separate files on the site (most of which are synthesis file) has tripled from 3804 to over 12000, which includes over 10000 object synthesis .RLE files.
  • Renamed files: Some existing page and pattern files have been renamed, for the sake of simplicity and consistency.
  • Renamed patterns: Some existing patterns have been renamed, to conform to popular usage.
  • Removed pages: Three pages have been removed, with their contents included directly in their parent pages: 13-bit pseudo-still-lifes, 15-bit P2 pseudo-oscillators, 16-bit P2 oscillators.
  • Added pages: Changes (this page), Search methodologies, Site map, Search for pattern (and search help), Links and other resources, Guns, Multi-colored Life, Alternate rules (including 2/2 Life, 3/4 Life, and Niemiec's rules), Surreal Numbers and Games (plus an interactive calculator for them).
  • All object lists: Stamps are in color, and font used to display cost in gliders improved. Many lists have additional objects. Color is used to categorize objects by ease of synthesis. Most stamps have object periods indicated in the top right corner, populations in the bottom left corner, and velocity (for moving patterns) or final population (for methuselahs) or page number (for large lists) in the bottom right corner. If objects of multiple periods, populations or velocities appear on one page, they are usually divided into sections by period, population, and/or velocity, and have similar indicators on each section.
  • All object pages: Many additional syntheses have been added; many existing syntheses have been improved or expanded; several incorrect syntheses have been fixed.
  • Object counts: Counts of 24-bit still-lifes and pseudo-still-lifes have been fixed (a few new items found). New computer search counts for for 21-bit period 2 oscillators and pseudo-oscillators. Computer verified search for 20-bit period 3 oscillators and pseudo-oscillators. All oscillators and pseudo-oscillators expanded to 21 bits (25 bits for periods 4 and higher, 24 bits for period 5 and 6 oscillators and period 15 pseudo-oscillators, and 22 bits for period 3 and 4 pseudo-oscillators).
  • Cost in gliders: Added new pages for objects constructible from 5 gliders, 6 gliders and 1 glider/bit, as well as naturally-occurring objects.
  • Objects sorted by size: Added new pages for objects of 21, 22, 23, 24, and 25 bits, one for larger objects, and one for not quite stable objects.
  • Objects sorted by type: Most sub-pages include new sub-sections for objects from 21-25 bits, larger objects, and costs in gliders.
  • Still-lifes: Added syntheses for all 14-bit and 15-bit* still-lifes, plus many larger ones. All still-lifes up to 17 bits can be synthesized. Added a huge list of all 18-bit ones.
  • Pseudo-still-lifes: Added syntheses for all 16-bit ones, and many larger ones. All pseudo-still-lifes up to 17 bits can be synthesized, as can most 18-bit and 19-bit ones. Added huge lists of all 17-bit and 18 -bit ones.
  • Period-2 oscillators: Added syntheses of all 18-bit* ones, and huge lists of all 19-bit, 20-bit, and 21-bit ones.
  • Period-2 pseudo-oscillators: Added syntheses for all 17-bit ones, and huge lists of all 18-bit, 19-bit, 20-bit, and 21-bit ones.
  • Oscillators by period: Added many new oscillator periods. Split out pseudo-oscillators by period into a separate page.
  • Oscillators and Pseudo-oscillators: These are now separated from oscillators by period. Individual periods now use one page for oscillators and a separate page for pseudo-oscillators. All oscillators and pseudo-oscillators are synthesized up to 21 bits* (and 25 bits for oscillators of period 8 and higher).
  • Oscillators: New sub-pages for periods 10, 12, 14, 24, 36, and 60.
  • Pseudo-oscillators: New sub-pages for periods 10, 12 and 60.
  • Spaceships: Added more spaceship velocities; separated exotic spaceships, flotillae and puffers (including wick-stretchers) into separate pages.
  • Constellations: Now include several useful artificial constellations, such as specialized eaters and reflectors.
  • Methuselahs: Now include final censuses, and pattern files to run methuselahs to completion.

(* NOTE: syntheses for 15-bit still-lifes, 18-bit period 2 oscillators, and 21-bit period 3 oscillators are currently incomplete, and are works in progress.)
