Due to the recent creation of this list, time has not yet permitted the syntheses of all these objects. About 40% of them have been synthesized so far. Objects with numbers above them in the stamp images have syntheses. Syntheses for the others have not yet been attempted, but will be added to this list as they are completed. These currently consist of two groups:

  • 141 beacon-on-inductor, all of which can be synthesized (using the same techniques that are used for smaller oscillators and still-lifes; e.g. if nothing else, all can be built from the corresponding 16-bit still-lifes block-on-same-inductor, and all 16-bit still-lifes can be synthesized.)
  • 150 miscellaneous oscillators that don't fit into any other category. Most can be synthesized (using the same techniques that are used for smaller oscillators and still-lifes). 97 could be built from smaller oscillators or still-lifes (that may not yet themselves have known syntheses), and only 45 defy existing technology.
